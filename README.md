# Tic-Tac-Toe

#Description
This project is a basic Tic-Tac-Toe, It's purpose is for me as a programmer to be able to learn how to use IIFE's.

Yes in this project I have learnt the idea of having private variables and functions.

#What did I use private Variables and Function?
The answer is, so that other parts of the code cannot be accessible to any person. Only "Public" methods and other parts of the code can be accessible by anyone.

For example in this Tic-Tac-Toe game, only game board, game results and text on the board can be seen by anyone who gets to play this game.
